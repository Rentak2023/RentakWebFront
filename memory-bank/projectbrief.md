# Project Brief

## Overview

Rentak appears to be a web application frontend for a rental property platform, based on the file structure showing components for landlords, tenants, units, and property management.

## Core Requirements

- Multi-language support (locale-based routing)
- Dashboard functionality for landlords and tenants
- Property listing and search capabilities
- Authentication system (login/register)
- Integration with various services (payment, location, etc.)

## Goals

1. Create intuitive user interfaces for property management
2. Implement robust search and filtering for property listings
3. Provide clear dashboards for different user roles
4. Ensure responsive and accessible design
5. Maintain clean code architecture and documentation
